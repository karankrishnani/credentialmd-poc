### Summary of Application
EverCred POC is a physician credentialing verification system featuring:

1. **Single Verification** - Verify individual physicians by NPI with real-time pipeline visualization
2. **Batch Verification** - Process up to 20 NPIs at once via text input or CSV upload
3. **Dashboard** - View verification metrics, cost tracking, and outcome distribution
4. **HITL Queue** - Human-in-the-loop review for escalated cases

### Technology Stack
- **Backend**: FastAPI + LangGraph + DuckDB + Python 3.11
- **Frontend**: Next.js 14 + TypeScript + Tailwind CSS + Recharts
- **Data Sources**: NPI Registry, CA DCA, OIG LEIE (all mocked for testing)

### Test Coverage
- 11 test NPIs covering all verification scenarios
- Mock mode for deterministic testing (EVERCRED_MOCK_MODE=true)
- Playwright browser automation for frontend testing
- curl/API testing for backend verification
